## 編譯方式

```sh
npm install

```

## 程式說明

程式啓動時，如果沒有找到本專案底下的"resources"，程式會開啓 userData 資料夾，這時請把"resources"複製過去，並重新開起程式。

## 角色視窗

可以使用 快捷鍵

shift+f1 放大

shift+f2 縮小

shift+f3 角色置頂

shift+f10 儲存設定

shift+f11 開啓資源目錄

shift+f12 控制台
